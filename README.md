# Harish Motamarri — Portfolio

Personal developer portfolio showcasing my projects, skills, and experience as a Full Stack Developer.

## Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, animations, dark/light theme
- **Vanilla JS** — No frameworks, zero dependencies
- **Google Fonts** — Playfair Display, JetBrains Mono, DM Sans

## Features

- 🌗 Dark / Light theme toggle with persistence
- 📱 Fully responsive (mobile-first)
- ⚡ Zero dependencies — pure HTML/CSS/JS
- 🎨 Custom cursor, scroll animations, marquee
- 📂 Accordion-style project case studies
- ♿ Accessible (ARIA labels, semantic HTML)

## Project Structure

```
Portfolio/
├── index.html              # Main HTML
├── css/
│   └── style.css           # All styles
├── js/
│   └── main.js             # All scripts
├── assets/
│   └── images/
│       └── profile.png     # Profile photo
├── .gitignore
└── README.md
```

## Run Locally

Open `index.html` in any browser, or use a local server:

```bash
npx -y http-server . -p 8080
```

Then visit `http://localhost:8080`.
